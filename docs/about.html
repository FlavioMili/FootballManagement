About the Project

Vision

Welcome to the Football Management Game Project, a passion-driven project built for fans who live and breathe the beautiful game. This is the opportunity for all fans to get a open-source alternative to other management games. Possibly, we want to have a better option with more personalisation and better gaming experience.

Core Development Principles

Behind the scenes, we follow key principles to ensure the game is solid, moddable, and fun to develop:

  • Community First: We’re building this for and with football fans and developers alike.
  • Clean Code: No shortcuts – just well-structured, readable, open-source code.
  • Modularity: Each system stands on its own – easy to expand, easy to maintain.
  • Scalability: Built to handle a single league or an entire football universe.
  • Data-Driven: Everything from match rules to player attributes needs to be configurable.

If you love football, strategy, or game development, you’re in the right place. Let’s build something legendary.

174/***************************************************************
175 * This is how to visualise the grid where to place the players:
176 *
177 * GK: (always separate, fixed role)
178 *
179 * Row 0 → furthest forward (attacking line)
180 * Row 4 → deepest outfield line (defensive line)
181 *
182 * Example 5×5 grid (row × col):
183 *
184 * (0,0) (0,1) (0,2) (0,3) (0,4)
185 * (1,0) (1,1) (1,2) (1,3) (1,4)
186 * (2,0) (2,1) (2,2) (2,3) (2,4)
187 * (3,0) (3,1) (3,2) (3,3) (3,4)
188 * (4,0) (4,1) (4,2) (4,3) (4,4)
189 *
190 * To place a player in a position, use:
191 * index = row * LINEUP_GRID_COLS + col;
192 *
193 * Example: Striker at (0,2) → index = 0 * 5 + 2 = 2
194 * Left-back at (4,0) → index = 4 * 5 + 0 = 20
195 *
196 ***************************************************************/

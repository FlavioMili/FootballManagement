150/***************************************************************
151 * This is how to visualise the grid where to place the players:
152 *
153 * GK: (always separate, fixed role)
154 *
155 * Row 0 → furthest forward (attacking line)
156 * Row 4 → deepest outfield line (defensive line)
157 *
158 * Example 5×5 grid (row × col):
159 *
160 * (0,0) (0,1) (0,2) (0,3) (0,4)
161 * (1,0) (1,1) (1,2) (1,3) (1,4)
162 * (2,0) (2,1) (2,2) (2,3) (2,4)
163 * (3,0) (3,1) (3,2) (3,3) (3,4)
164 * (4,0) (4,1) (4,2) (4,3) (4,4)
165 *
166 * To place a player in a position, use:
167 * index = row * LINEUP_GRID_COLS + col;
168 *
169 * Example: Striker at (0,2) → index = 0 * 5 + 2 = 2
170 * Left-back at (4,0) → index = 4 * 5 + 0 = 20

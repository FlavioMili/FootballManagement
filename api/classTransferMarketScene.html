Detailed Description

Full-screen ImGui scene for the transfer market.

Layout: Top bar (budget + window status), tab bar, content area. Tabs: Buy Players, Your Listings, Incoming Bids. Dialogs: Confirmation (buy/sign), Counter-offer.
